How it works: Upload any photo and the server runs three XAI methods on a ResNet-50 model: RISE randomly masks the image and measures prediction drop, LIME fits a local linear model around superpixel perturbations, and GradCAM propagates class gradients back to the last conv layer. Each method is scored with Deletion AUC ↓ (lower = better: important pixels identified efficiently) and Insertion AUC ↑ (higher = better: prediction recovers faster with fewer pixels).
